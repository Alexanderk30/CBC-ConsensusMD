# Impeccable — ConsensusMD

This file is the single source of truth for design decisions on the ConsensusMD frontend. Any design skill (craft, layout, polish, delight, etc.) should read this before generating code.

## Design Context

### Users
The frontend is built to be filmed, not used. The primary viewer is a hackathon judge watching a short demo video of the system in action — they will not click anything, they will not resize the window, they will not scroll. They will watch agents debate a clinical case on screen for a minute or two and decide whether this team shipped something credible.

This is **not** a clinician-facing product in the hands-on sense, despite the README positioning. It is a cinematic showpiece that *looks like* a serious clinician-facing instrument. Both readings have to hold: the judge must believe a real doctor could sit down at this and use it, while also being visually arrested enough to remember it an hour later among twenty other demos.

Viewing context: fixed 1600px-wide layout (per `Caduceus.html` viewport meta), played back as video at variable sizes. No interaction budget. Every frame must read as a still.

### Brand Personality
**Anatomical. Surgical. Philanthropic.**

- **Anatomical** — drawn from the tradition of medical illustration, not dashboard UI. Proportions, curves, and labels behave like a Gray's Anatomy plate: considered, annotated, reverent of the subject. The caduceus motif is not decoration, it's the conceptual spine.
- **Surgical** — precise, controlled, sharp. Nothing wasted. Typography is tight; lines are thin; spacing is deliberate. When the antagonist challenges, it should feel like an incision, not an explosion.
- **Philanthropic** — this balances surgical coldness. The system exists to help a patient somewhere, not to flex technical sophistication. Warmth comes through the paper palette (never pure white), the serif body voice (Fraunces), and restraint (nothing is screaming for attention).

### Aesthetic Direction
The Caduceus design handoff (`design_handoff_caduceus/reference/`) is the locked visual foundation. Do not redesign it. Extend, refine, and polish within its established language.

**Theme**: Light only. Warm paper base (`oklch(0.96 0.006 85)`), deep teal ink (`oklch(0.22 0.030 210)`), never pure white, never pure black.

**Type stack** (locked):
- Display/body serif: **Fraunces** — with `font-feature-settings: 'ss01'` for the anatomical character
- Label/meta/data monospace: **JetBrains Mono** — 9–11px, 0.15em+ tracking, uppercase
- UI sans: **Inter** — used sparingly; Fraunces does most of the body work

Fraunces is on the banned reflex-fonts list in the impeccable skill spec. The handoff predates that guidance and the whole visual system is built around it. Keep it. Do not substitute.

**Color system** (locked tokens, use CSS variables — don't invent hex):
- `--ink-0` through `--ink-3`: paper surfaces, light to slightly tinted
- `--bone-0` through `--bone-3`: deep teal ink, primary → tertiary text
- `--artery`: muted terracotta. Reserved for the antagonist agent and adversarial moments (challenges, dissent). Never decorative.
- `--vein`: steel blue. Secondary accent.
- `--ichor`: muted sage. Reserved for consensus / resolution / verdict.

**Motifs**: the caduceus staff as central crest; thin dashed rotating rings around agent nodes; a sustained terracotta ring that wraps the specialist node OPHIS is challenging in any given round (single signal, single place to look — replaces an earlier per-challenge debate-arc visualization); 1px hairline borders; film-plate restraint.

**Composition**: asymmetric HUD panels overlaid on a central live scene. No card grids. No centered hero-metric layouts. Absolute positioning with intentional negative space is correct for this project, not a smell.

### Anti-references
- AI dashboards (tile grids of KPI cards, purple-to-blue gradients, neon cyan accents) — this is the single most important thing this interface must never look like. The whole handoff is an argument against it.
- Medical SaaS onboarding UI (rounded rectangles, friendly illustrations, stock doctor photos) — wrong tradition. The reference tradition is printed medical illustration, not B2B SaaS.
- Gradient text — already forbidden by the impeccable skill; doubly wrong here.
- Side-stripe borders as decorative accents — forbidden by the skill. The handoff uses `border-left` *semantically* on utterance bubbles (indicating speaker), which is allowed because it carries meaning, not mood. Do not extend this pattern decoratively to cards, panels, or list items.

## Design Principles

1. **Every frame reads as a still.** The primary artifact is a video, so no element may depend on motion to be legible. Motion is an amplifier of a composition that already works static, never a crutch for one that doesn't.
2. **Credibility is earned by restraint; spectacle is earned by a single arresting moment.** Do not try to impress by adding. Impress by subtracting until what remains is undeniable, then spend budget on *one* cinematic gesture per scene (the antagonist's strike, the consensus bloom, the convergence moment).
3. **Color carries meaning, not mood.** Terracotta means antagonist. Sage means consensus. Teal ink means the paper's voice. Never use an accent color "to add visual interest" — that breaks the diagnostic legibility of the whole system.
4. **Work anatomically, not geometrically.** Given the choice between a grid alignment and an anatomically-reasoned placement (e.g., agents arrayed as they would sit around a consultation, not as a card grid), choose anatomical. The caduceus spine, the agent-node ring system, the crest — these are the load-bearing structures, not a 12-column grid.
5. **Surgical type discipline.** Fraunces for body and utterance, JetBrains Mono for labels/meta/data only, Inter only where neither fits. Uppercase + wide tracking is for labels, never for headlines or body. 1.25+ ratio between type-scale steps; no flat hierarchies.
6. **Patient-first voice.** Copy names the patient as the subject, not the system. The antagonist exists to protect the patient from diagnostic anchoring — never framed as combat, aggression, or spectacle against the specialists. Deadlock is uncertainty surfaced, not system failure. The humanitarian mission (clinicians working without a specialist down the hall) is named explicitly in at least one high-visibility surface, not left as subtext. When in doubt between a line that reads as *clever* and a line that reads as *humane*, choose humane.
